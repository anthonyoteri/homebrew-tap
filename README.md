# homebrew-tap

Homebrew formulae for [anthonyoteri](https://github.com/anthonyoteri)'s tools.

## Usage

```sh
brew tap anthonyoteri/tap
```

## Formulae

### structprop-validator

A validator for [structprop](https://github.com/anthonyoteri/serde-structprop) configuration files.

```sh
brew install anthonyoteri/tap/structprop-validator
```

**Platforms:** macOS (Apple Silicon), Linux (x86_64)

For Windows, download the binary directly from the
[releases page](https://github.com/anthonyoteri/serde-structprop/releases),
or install via `cargo`:

```sh
cargo install structprop-validator
```
